1. Présentation du projet

Imagine un entrepôt rempli de caisses éparpillées. Tu es le magasinier. Ta mission : pousser chaque caisse jusqu'à son emplacement marqué au sol. Tu ne peux que pousser, jamais tirer. Une caisse coincée dans un angle, c'est raté — la partie est perdue.

Ce casse-tête s'appelle le Sokoban (倉庫番, « gardien d'entrepôt »), inventé en 1982. Il fascine les informaticiens : certains niveaux font partie des problèmes les plus durs qui existent mathématiquement. Et maintenant, tu vas le coder en Python, de A à Z.

1.1 Essaie par toi-même — avant de coder

Joue quelques instants avec les touches fléchées ou ZQSD. Repère ce qui bloque, ce qui se passe quand tu pousses une caisse. C'est ce comportement que tu vas programmer.

Touches : ↑↓←→ ou Z/Q/S/D  |  Coups : 0  |

2.1 La classe Niveau

Cours — Classe, constructeur, encapsulation

En Python, __init__ est le constructeur. self désigne l'instance. Un attribut préfixé par _ est encapsulé : c'est une convention signalant qu'il ne doit pas être modifié directement depuis l'extérieur.

class Niveau:
    def __init__(self, grille_texte):
        self.grille    = [list(l) for l in grille_texte]
        self.nb_lignes = len(self.grille)
        self.nb_cols   = max(len(r) for r in self.grille)
        self.cibles    = self._trouver_cibles()  # méthode « privée »

    def _trouver_cibles(self):
        return [(l, c)
                for l in range(self.nb_lignes)
                for c in range(len(self.grille[l]))
                if self.grille[l][c] in ('.', '*', '+')]

2.4 Héritage — NiveauChrono

Cours — Héritage, super() et polymorphisme

L'héritage permet à une classe fille d'hériter de tous les attributs et méthodes d'une classe mère, et d'en ajouter ou redéfinir (surcharger). On appelle le constructeur parent avec super().__init__(...). 

class Animal:
    def __init__(self, nom): self.nom = nom
    def crier(self): return "..."

class Chien(Animal):
    def crier(self): return "Ouaf !"   # surcharge

rex = Chien("Rex")
print(rex.nom)     # hérité de Animal → "Rex"
print(rex.crier()) # surchargé        → "Ouaf !"

2.5 Récursivité et preuve de terminaison

Cours — Récursivité, variant et terminaison

Une fonction récursive se termine si et seulement si un variant — une quantité entière positive — décroît strictement à chaque appel. Pour nb_caisses_placees(grille, idx), le variant est len(grille) - idx.

Pile d'appels pour une grille de 3 lignes

nb_caisses(g, 0) ← actif
nb_caisses(g, 1)
nb_caisses(g, 2)
cas de base : idx == 3 → return 0

Variant : 3 - idx

idx=0 → 3   idx=1 → 2   idx=2 → 1

idx=3 → 0 → cas de base

⚠️ Python limite la récursion à 1 000 niveaux par défaut (sys.getrecursionlimit()).

2.6 Pile explicite — Historique des coups

Cours — Pile LIFO en Python

Une pile (LIFO — Last In, First Out) : append() empile, pop() dépile. Exemple concret : l'historique d'annulation (Ctrl+Z) dans tout éditeur de texte. 

pile = []
pile.append("état_1")   # sommet → état_1
pile.append("état_2")   # sommet → état_2
dernier = pile.pop()     # retire état_2 → "état_2"

Pour sauvegarder un état de grille sans qu'il soit modifié ensuite : import copy; copy.deepcopy(grille).

4.2 BFS — Parcours en largeur avec table d'états

Cours — BFS et File FIFO — Complexité O(V+E)

Le BFS explore le graphe couche par couche avec une file FIFO (collections.deque). Il garantit le chemin le plus court dans un graphe non pondéré.

4.3 DFS — Parcours en profondeur

Cours — DFS et pile LIFO — Complexité O(V+E)

Le DFS s'enfonce le plus loin possible avec une pile LIFO (list). Contrairement au BFS, il ne garantit pas le chemin le plus court.

Comparaison : BFS → file FIFO → chemin court   DFS → pile LIFO → exploration complète   Les deux : O(V+E)

4.4 Programmation dynamique — Rendu de monnaie

Cours — Mémoïsation (top-down) vs Tabulation (bottom-up)

La prog. dynamique décompose un problème en sous-problèmes qui se recoupent, et mémorise les résultats pour ne pas les recalculer.

# Top-down : récursif + cache
cache = {}
def rendu_memo(somme, pieces):
    if somme == 0: return 0
    if somme in cache: return cache[somme]
    mini = min(1 + rendu_memo(somme-p, pieces) for p in pieces if p <= somme)
    cache[somme] = mini
    return mini

# Bottom-up : table itérative  — O(somme × nb_pièces)
def rendu_tab(somme, pieces):
    dp = [float('inf')] * (somme + 1)
    dp[0] = 0
    for s in range(1, somme+1):
        for p in pieces:
            if p <= s: dp[s] = min(dp[s], 1 + dp[s-p])
    return dp[somme]

Exercice 4.4 — Rendu de monnaie : mémoïsation ET tabulation Intermédiaire

Indice

La boucle de jeu est gérée automatiquement par arcade.run(). Elle appelle on_draw() à chaque rafraîchissement et on_key_press() à chaque appui.

Cours — Système de coordonnées Arcade

Arcade utilise un repère mathématique standard : l'origine (0, 0) est en bas à gauche, l'axe Y monte vers le haut. C'est l'inverse de notre grille Sokoban (où la ligne 0 est en haut). La conversion est : 

# Grille Sokoban (ligne 0 = haut) → repère Arcade (y=0 = bas)
def coords_arcade(ligne, col, taille, nb_lignes):
    x = col * taille + taille / 2                       # centre horizontal
    y = (nb_lignes - 1 - ligne) * taille + taille / 2  # axe Y inversé
    return x, y

La fonction arcade.LBWH(left, bottom, width, height) définit un rectangle par son coin bas-gauche.

Indice — Inversion de l'axe Y

# ── HISTORIQUE (Pile LIFO — B3) ────────────────────────────────────────
import copy

class Historique:
    """Pile LIFO pour l'annulation des coups (Ctrl+Z).
    Connecte la structure pile enseignée en 2.6 à la vue Arcade."""

    def __init__(self):
        self._pile = []   # encapsulation : accès uniquement via méthodes

    def empiler(self, etat):
        """Sauvegarde une copie profonde de l'état courant."""
        self._pile.append(copy.deepcopy(etat))

    def depiler(self):
        """Retourne le dernier état ou None si vide."""
        return self._pile.pop() if self._pile else None

    def est_vide(self):
        return len(self._pile) == 0

    def taille(self):
        return len(self._pile)

class Niveau:
    def __init__(self, grille_texte):
        self.grille    = [list(l) for l in grille_texte]
        self.nb_lignes = len(self.grille)
        self.nb_cols   = max(len(r) for r in self.grille)

                          for li in range(self.nb_lignes)
                          for c  in range(len(self.grille[li]))
                          if self.grille[li][c] in ('.', '*', '+')]

class Joueur:
    def __init__(self, ligne, col):
        self.ligne = ligne; self.col = col; self.nb_coups = 0

    def deplacer(self, direction, niveau):
        """Déplace le joueur. Retourne True si OK, False sinon.
        A3 — vérification des bornes avant tout accès à la grille."""
        dl, dc = VECTEURS[direction]
        nl, nc = self.ligne + dl, self.col + dc
        g = niveau.grille
        # A3 — bornes vérifiées AVANT l'accès : évite l'index négatif silencieux
        if not (0 <= nl < len(g) and 0 <= nc < len(g[nl])):
            return False
        if g[nl][nc] == '#':
            return False
        if g[nl][nc] in ('$', '*'):
            nl2, nc2 = nl + dl, nc + dc
            # A3 — bornes vérifiées pour la case derrière la caisse
            if not (0 <= nl2 < len(g) and 0 <= nc2 < len(g[nl2])):
                return False
            if g[nl2][nc2] == '#' or g[nl2][nc2] in ('$', '*'):
                return False
            g[nl2][nc2] = '*' if g[nl2][nc2] == '.' else '$'
            g[nl][nc]   = '+' if g[nl][nc]   == '*' else '@'
        else:
            g[nl][nc] = '+' if g[nl][nc] == '.' else '@'
        g[self.ligne][self.col] = '.' if g[self.ligne][self.col] == '+' else ' '
        self.ligne, self.col = nl, nc
        self.nb_coups += 1
        return True

class Partie:
    def __init__(self, niveau, joueur):
        self.niveau = niveau; self.joueur = joueur

    def est_gagne(self):
        """O(L×C) — appelé UNE SEULE fois par déplacement (A4)."""
        return all('$' not in ligne for ligne in self.niveau.grille)

            return

            return

            return

            self._aller_menu()
            return

        y = (self.niveau.nb_lignes - 1 - idx_l) * TAILLE + TAILLE / 2 + HUD

5.5 Animation et boucle de jeu — on_update()

Cours — on_update(delta_time) et animation

La méthode on_update(delta_time) est appelée environ 60 fois par seconde. delta_time est le temps écoulé depuis le dernier appel (en secondes, typiquement ≈ 0.016 s). Elle permet de faire évoluer l'état du jeu indépendamment de l'affichage — c'est le principe fondamental du MVC appliqué à la boucle de jeu. 

def on_update(self, delta_time):

        self._flash_timer += delta_time
        if self._flash_timer >= 0.5:   # toutes les 0.5s
            self._flash_timer   = 0.0
            self._flash_visible = not self._flash_visible

5.6 Sélecteur de niveaux avec arcade.View

Arcade propose un système de vues pour gérer plusieurs écrans dans la même fenêtre (menu, jeu, victoire…). Chaque vue hérite de arcade.View et redéfinit ses propres on_draw() et on_key_press(). Le passage d'une vue à l'autre se fait avec self.window.show_view(nouvelle_vue).

Question 3 — Détecter les coins morts (deadlock simple) Avancé

Un coin mort est une position où une caisse ne pourra jamais être poussée sur une cible, quelles que soient les actions du joueur. La détection la plus simple : une caisse dans un angle (deux murs perpendiculaires adjacents) sur une case qui n'est pas une cible.

Écris la fonction est_coin_mort(grille, ligne, col) qui retourne True si la case (ligne, col) est un coin mort pour une caisse.

Indice — Définition d'un angle

Un angle : soit (mur au nord ET mur à l'ouest), soit (mur au nord ET mur à l'est), soit (mur au sud ET mur à l'ouest), soit (mur au sud ET mur à l'est). Et la case elle-même n'est pas une cible ('.' ).

Correction 
def est_coin_mort(grille, ligne, col):
    if grille[ligne][col] in ('.', '*', '+'):
        return False   # une cible n'est jamais un coin mort
    def mur(l, c):
        if l < 0 or l >= len(grille): return True
        if c < 0 or c >= len(grille[l]): return True
        return grille[l][c] == '#'
    # Vérifier les 4 paires d'angles
    nord  = mur(ligne-1, col)
    sud   = mur(ligne+1, col)
    ouest = mur(ligne, col-1)
    est   = mur(ligne, col+1)
    return (nord or sud) and (ouest or est)
